# Mandelbrot Explorer

A GPU accerelated Mandelbrot explorer implemented in Java 8 using aparapi (https://gitter.im/Syncleus/aparapi)

Use mouse to explore:

- scroll wheel to zoom
- left click to move
- [Control] double click to to [un-]zoom
